## 项目背景
```
该项目背景为年会活动中，员工访问项目，输入自己姓名并上传照片，再点击生成海报，便可生成图片海报保存到手机中。同时也可分享给好友或分享至朋友圈
```

## 项目简介
基于 vue-cli;  

生成海报：html2canvas;  

分享链接：[微信公众号开发相关内容](https://developers.weixin.qq.com/doc/offiaccount/OA_Web_Apps/JS-SDK.html)
